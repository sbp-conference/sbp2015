Downloading GDELT (Data Chellenge)

Daily Event Dataset
This data is provided as a comma-separated value (CSV) dataset. Each row in the file is an event, and the columns correspond to the guide from GDELT, available here. To obtain the daily event dataset, visit the following page. Here, you can download daily updated files from April 1, 2013.

Historical Event Dataset
This dataset is another CSV of historical events (those that happened from 1979 - April 1, 2013). To get historical event data, visit here. Note: The columns here are slightly different than above. For the column guide for this dataset visit here.

Global Knowledge Graph
Provided in CSV format, it can be downloaded here. For information about the columns in the dataset and some general background about the dataset overall, you can read this cookbook. Note: Each day produces two files: the counts file and the graph file.
